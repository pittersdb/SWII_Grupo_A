- Herramienta para abrir el archivo del diagrama entidad-relacion
	
	Dentro de la carpeta "others/Diagrama en MySQL workbench" se encuentra un archivo .mwb que debe ser abierto con la herramienta MySQL Workbench que puede ser descargado de aqui http://www.mysql.com/products/workbench/

- Herramienta para abrir el diagrama de clases UML
	
	Dentro de la carpeta "others/Diagrama de clases UML" se encuentra un archivo llamada SysMedicUML.umlj que debe ser abierto con la herramienta Star Uml version 2 que puede ser descargada de aqui http://staruml.io/

- Herramienta para abrir el diagrama de componentes
	
	Dentro de la carpeta "others" se encuentra un archivo llamada diagramaComponentes.diaFile que debe ser abierto con la herramienta DIA que puede ser descargada de aqui http://dia-installer.de/
